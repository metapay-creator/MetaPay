# MetaPay  
A Web3-based Circular Basic Income Simulator

## Problem Statement  
Most basic income proposals fail due to:
- Unsustainable funding models  
- Heavy reliance on taxation  
- Public distrust of centralized distribution  

These failures hinder civic engagement and leave vulnerable populations without scalable solutions.

## Solution Overview  
MetaPay is a self-sustaining basic income simulator using Web3 wallets and smart contracts.  
It distributes ₩500,000 (~$370) monthly to 50 million users, automatically reclaims 10% of expenditures for continuous circulation.  
This removes the need for new taxes and enhances civic engagement and economic activity.

## Core Features  
- **Smart Contract Automation**: Transparent distribution and automatic recollection  
- **AI-Powered Economic Simulation**: Real-time adaptive policy modeling  
- **Immutable Ledger**: EVM-compatible tracking of all actions  
- **Gamification**: Wallet rewards, airdrops, engagement levels  

## Technologies Used  
- Web3 wallets (MetaMask, Hyperion)  
- Smart contracts (EVM-compatible)  
- AI Policy Simulator (Python ML)  
- No-code UX prototype (Figma)

## Community Engagement  
- Simulate receiving and spending MetaPay through a user interface  
- Earn points by connecting wallets, spending, and inviting others  
- Contribute to feedback loops and policy suggestions
